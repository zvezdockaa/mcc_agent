# MCC AI Agent

ИИ-агент для автоматического определения MCC-кода и выдачи рекомендаций по продажам эквайринговых продуктов.

## Возможности

- 🔍 Поиск организаций через 2GIS API
- 🤖 Автоматическое определение MCC-кодов
- 💬 Обратная связь через Telegram
- 📝 Сохранение истории запросов
